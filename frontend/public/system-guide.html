AI 处理管道

系统采用两阶段 AI 策略：快速小模型处理海量新闻，大型视觉语言模型生成战场综合报告，全部在本地 Ollama 运行，无需联网 API。

⚡ deepseek-r1:1.5b — 快速处理
  • 📌 用途：新闻摘要翻译、分类、地理坐标提取
  • ⏱ 响应：约 3 秒/条
  • 📦 模型大小：1.1 GB
  • 🔑 特性：CoT 在 <think> 标签内，content 字段包含实际 JSON 输出
  • 📊 token 预算：700
  • 🌐 中文能力：直接输出中文摘要，置信度分级
🧠 qwen3-vl:8b — 深度分析
  • 📌 用途：战场综合报告生成 + 图像军事分析
  • ⏱ 响应：约 60-120 秒/次（8192 token）
  • 📦 模型大小：6.1 GB
  • 🔑 特性：CoT 在 Ollama thinking 字段，content 为空需 8192 token 预算
  • 👁 视觉：支持 base64 图像输入，分析军事装备和场景
  • 📊 超时：480s，防止报告生成超时失败

▌ 战场报告生成（generate_daily_summary）
输入：过去 24h 新闻摘要（最多 50 条） + 事件列表（最多 30 条）

模型：qwen3-vl:8b，num_predict=8192，timeout=480s

输出 JSON：
{
  "summary":          "300-400字综合态势分析",
  "intensity_score":  7.5,                    // 0-10 冲突烈度
  "key_developments": ["要点1", "要点2", ...],
  "hotspots": [
    {"name": "霍尔木兹海峡", "lat": 26.58, "lon": 56.45, "score": 8.0, "reason": "美伊对峙..."}
  ],
  "outlook": "50字未来态势研判"
}

坐标校验：hotspot lat/lon 自动校验中东范围（纬度10-45°，经度32-75°）
          若为0,0则按50+关键词查表修正（纳坦兹/德黑兰/霍尔木兹等）
